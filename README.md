# EnglishCard — Telegram-бот для изучения английского языка

## Описание
Telegram-бот для изучения английских слов с использованием PostgreSQL.

## Функционал
- Вопросы с 4 вариантами ответа
- Добавление слов пользователем
- Удаление слов (персонально)
- Хранение данных в PostgreSQL

## Установка
1. Клонировать репозиторий
2. Создать виртуальное окружение
3. Установить зависимости:
   pip install -r requirements.txt
4. Создать базу данных PostgreSQL
5. Выполнить `db.sql`
6. Создать `.env` файл
7. Запустить `python main.py`
